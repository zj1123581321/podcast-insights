关于本站 · 创作历程

这是把播客《肥话连篇》全部 234 期里的 935 条推荐聚成的一张可查、可逛的地图。缘起很简单:听播客时主播常推荐些东西——某个城市的馆子、在追的剧、顺手的小物件,当下记一句"回头试试",事后却无从检索(音频里的文本搜不到)。于是干脆让手头的工具自己长出了个解法。

怎么做出来的

  1. 把声音变成字。先抓下全部 234 期链接(小宇宙网页端默认只显示最近 10 集,得专门抓历史列表),提交到自建的音视频转录服务,得到区分说话人、经 LLM 校对的文字稿。ASR 常把主播名写错,额外做了一步人名归一。
  2. 把字变成结构化数据。用 Claude Code 的 Workflow 对 234 篇稿子跑同一套提取逻辑(Sonnet,约 8 并发,三四十分钟跑完),抠出"推荐了什么"——实地 / 好物 / 影视剧,各有统一字段。每条都做了原文反查:确认不是模型编的,也能跳回原话听上下文。
  3. 把数据变成地图。城市级聚合落点;具体店铺点一下店名跳高德搜索,精确定位甩给高德。底图特意用了带审图号的合规版本。整个可视化部署在 Cloudflare 上。
